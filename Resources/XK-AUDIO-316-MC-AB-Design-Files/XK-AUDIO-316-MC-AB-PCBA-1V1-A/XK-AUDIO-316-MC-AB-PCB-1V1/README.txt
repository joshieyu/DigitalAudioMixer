README.TXT
----------

Additional information relating to the build of design XMS0004 (xcore.ai MC audio board).

Design Ref    : XMS0004
Version       : 1V1

Date          : 17th August 2022

PCB Design tool : Altium Designer 21

Description of files included in XMS0004-PCB-1V1.ZIP:

./README.TXT    This file.

./Gerber/       This directory contains the following gerber files for the design.

XMS0004-1V1.REP       Gerber Report File.
XMS0004-1V1.GTL       Layer 1 (Top) Copper
XMS0004-1V1.GP1       Layer 2 Copper
XMS0004-1V1.GP2       Layer 3 Copper
XMS0004-1V1.GBL       Layer 4 (Bottom) Copper
XMS0004-1V1.GTO       Silkscreen Top
XMS0004-1V1.GBO       Silkscreen Bottom  
XMS0004-1V1.GTS       Solder Mask Top
XMS0004-1V1.GBS       Solder Mask Bottom
XMS0004-1V1.GTP       Solder Paste Top
XMS0004-1V1.GBP       Solder Paste Bottom
XMS0004-1V1.GM13      Top Assembly Data
XMS0004-1V1.GM14      Bottom Assembly Data
XMS0004-1V1.GM17      Fabrication Instructions
XMS0004-1V1.GD1       Drill Diagram.

All gerbers are RS-274-X format, metric (0000.000), no zero suppression, absolute coordinates.

./Drill/        This directory contains the following drill data files for the design.

XMS0004-1V1-RoundHoles-Plated.TXT       Plated Through Round Holes NC Drill File
XMS0004-1V1-RoundHoles-NonPlated.TXT    Non Plated Through Round Holes NC Drill File
XMS0004-1V1-SlotHoles-Plated.TXT        Plated Through Slot Holes NC Drill File
XMS0004-1V1.DRR                         NC Drill Report (Description of Files, Drill Sizes & Quantities)

NC Drill files are Excellon Format, metric (0000.000), no zero suppression, absolute coordinates.

./Netlist/      This directory contains the netlist file for equivalence checking and bare board test.

XMS0004-1V1.IPC       IPC-D-356 Format Netlist

./ODB++/        This directory contains an ODB++ file for the design.

XMS0004-1V1.ZIP       The design in ODB++ Format.
